DSL
===

The module :mod:`hymn.dsl` provides types and functions from other modules of
this package, so that they can be imported all at once easily.

Python

.. code-block:: python

  from hymn.dsl import *

Hy

.. code-block:: clojure

  (import hymn.dsl [*])
